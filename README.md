# mymemo

간단한 메모를 구현하는 앱

## 주요 기능

-Floating 버튼을 통해 할 일들을 추가함  
-삭제해야되는 일이 생겼을 땐 Floating 페이지로 넘어가 상단의 Delete 아이콘을 눌러 삭제함  
-우선순위가 되는 할 일들은 핀 버튼을 눌러 상단에 배치함
